Apa itu Bootstrap 4?

Bootstrap adalah framework CSS untuk membuat tampilan web. Bootstrap menyediakan class dan komponen yang siap dipakai, sehingga kita tidak perlu menulis kode CSS dari nol. Bootstrap awalnya dikembangkan oleh developer Twitter, lalu dibuat open source agar semua orang dapat berkontribusi di dalamnya.

Pada saat ini hampir semua web developer telah menggunakan bootstrap untuk membuat tampilan front-end menjadi lebih mudah dan lebih cepat, karena kita hanya perlu menambahkan class-class tertentu untuk misalnya membuat tombol, grid, navigasi dan lainnya.

Perbedaan tampilan sangat terlihat jelas, apabila kita membuat form tanpa bootstrap terlihat berkesan jadul dan tidak menarik,. Selain itu tampilan formnya pun nantinya tidak akan responsive seperti apabila kita menggunakan bootstrap.
